# 115 磁力推送 Bot

## 功能介绍

Telegram Bot，用于绑定 115 网盘账号后，直接在聊天中发送磁力链接即可添加到 115 离线下载。支持扫码绑定、自动刷新 Token、后台定时续期。

**主要功能：**
- 🧲 发送磁力链接自动添加到 115 离线下载
- 📱 扫码绑定 115 账号
- 🔄 自动刷新 Token，无需手动续期
- 🔓 `/unbind` 解绑账号

## 快速部署

### Docker Run

```bash
docker run -d \
  --name 115-bot \
  --restart unless-stopped \
  -e TZ=Asia/Shanghai \
  -e BOT_TOKEN=你的Bot_Token \
  -e CLIENT_ID=100195135 \
  -e ADMIN_IDS=123456789 \
  -e LOG_LEVEL=INFO \
  -v ./data:/app/data/ \
  -v ./user_tokens:/app/user_tokens/ \
  aiastia/mytgbot:115
```

### Docker Compose

```yaml
services:
  115-bot:
    image: aiastia/mytgbot:115
    restart: unless-stopped
    environment:
      - TZ=Asia/Shanghai
      - BOT_TOKEN=你的Bot_Token
      - CLIENT_ID=100195135
      - ADMIN_IDS=123456789
      - LOG_LEVEL=INFO
    volumes:
      - ./data:/app/data/
      - ./user_tokens:/app/user_tokens/
    logging:
      options:
        max-size: "10m"
        max-file: "3"
```

```bash
docker compose up -d
```

## 环境变量

| 变量 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| `BOT_TOKEN` | ✅ | - | Telegram Bot Token |
| `CLIENT_ID` | ❌ | `100195135` | 115 客户端 ID |
| `ADMIN_IDS` | ❌ | - | 管理员 ID（逗号分隔） |
| `LOG_LEVEL` | ❌ | `INFO` | 日志级别 |
